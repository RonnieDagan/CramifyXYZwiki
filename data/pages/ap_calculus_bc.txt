**Continuity**

A function f is continuous at â€œcâ€ if:

  * The value exists- The value of the function is defined at â€œcâ€ and f(c) exists

  * The limit exists - The limit of the function must exist at â€œcâ€
    * The left and right limits must equal

  * Function=limit. The value of the function at â€œcâ€ must equal the value of the limit at â€œcâ€

**Discontinuity**

  * **Removable** â†’ discontinuity at â€œcâ€ is called removable if the function can becontinuous by defining f(c)

  * **Non-removable** â†’ discontinuity at â€œcâ€ is called non-removable if the function cannotbe made continuous by redefining f(c)

**Differentiability**

**In order for a function to be differentiable at x = c:**

  * The function must be continuous at x = c

  * Its **left and right** derivative must equal each other at x = c

**Related Rates**

What is the purpose of related rates?

  * The purpose is to find the rate where a quantity changes

  * The rate of change is usually with respect to time

How to solve it?

  - Find an equation that ties your variables together.
